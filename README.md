# Summer School Schedule - July 2026

An interactive schedule and calendar tool for the "Summer School" program (בית הספר של החופש הגדול) for grades 4, 5, and 6 (כיתות ד', ה', ו').

## 🌐 Live Website
The interactive calendar is hosted live at:
👉 **[https://pinimatz.github.io/summerschool/](https://pinimatz.github.io/summerschool/)**

## 📁 Repository Contents
*   `index.html`: The self-contained interactive web application.
*   `לוח שיעורים יולי כיתות ד ו-ו v3.xlsx`: Excel sheet with student attendance highlights.
*   `לוח שיעורים יולי - כל הקבוצות.xlsx` / `pdf`: Public spreadsheet calendar for all groups.
*   `לוח שיעורים אינטראקטיבי - קוד לגוגל קולאב.py`: Python code for running the schedule query system on Google Colab.
